6. バグ修正（2件）

6.1 MediaProcessor: private_class_method 配置ミス

問題: private_class_methodがメソッド定義の前に配置されており、テスト環境（eager_load: false）でNameErrorが発生。

修正: メソッド定義の後に移動。

6.2 AudioInterviewService: 未定義定数参照

問題: STTClient::MAX_FILE_SIZEが未定義。

修正: Rails.application.config.interview.stt_max_file_sizeに変更。

セキュリティ機能

  • ホワイトリスト: localhost（127.0.0.1, ::1）は制限除外

  • 429レスポンス: JSON + Retry-After, X-RateLimit-* ヘッダー
  • 403レスポンス: ブロックIPに対するJSON形式レスポンス
